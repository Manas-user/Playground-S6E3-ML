# 🧠 Kaggle Playground Series S6E3 - Machine Learning Project

## 📌 Overview

This project is based on the Kaggle Playground Series (Season 6, Episode 3).
The objective is to build a machine learning model to predict the target variable using structured tabular data.

---

## 🚀 Workflow

1. Data Loading (Kaggle dataset)
2. Data Cleaning & Preprocessing

   * Handling missing values
   * Encoding categorical variables
3. Feature Engineering
4. Model Training

   * Random Forest
   * XGBoost
5. Model Evaluation

   * Train/Test Split
   * Cross Validation
6. Submission File Generation

---

## 🧠 Models Used

* Random Forest Classifier
* XGBoost Classifier

👉 Tree-based models perform best on tabular datasets.

---

## 📊 Evaluation

* Accuracy Score used for validation
* Cross-validation for robustness

---

## 📁 Project Structure

* `notebooks/` → Kaggle notebook
* `submission/` → Final predictions
* `requirements.txt` → Dependencies

---

## ⚙️ Installation

```bash
pip install -r requirements.txt
```

---

## ▶️ How to Run

1. Open notebook in Kaggle / Jupyter
2. Run all cells
3. Generate `submission.csv`

---

## 📌 Key Learnings

* Importance of preprocessing in real-world datasets
* Feature engineering impact on performance
* Tree-based models outperform linear models on tabular data
